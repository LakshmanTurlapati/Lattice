    Create Lattice-owned result and validation modules.

    In `results/errors.ts`:
    - Export `ValidationIssue` with readonly `message: string` and optional readonly `path?: readonly (string | number | symbol)[]`.
    - Export `ValidationError` with readonly `kind: "validation"`, `message: string`, optional readonly `output?: string`, and readonly `issues: readonly ValidationIssue[]`.
    - Export `ExecutionUnavailableError` with readonly `kind: "execution_unavailable"` and readonly `message: string`.
    - Export `LatticeRunError = ValidationError | ExecutionUnavailableError`.

    In `results/result.ts`:
    - Import `type { ArtifactRef }`, `type { ExecutionPlanStub }`, `type { OutputContractMap }`, `type { InferOutputMap }`, and `type { LatticeRunError }`.
    - Export `RunSuccess<TOutputs extends OutputContractMap>` with readonly `ok: true`, readonly `outputs: InferOutputMap<TOutputs>`, readonly `artifacts: readonly ArtifactRef[]`, and readonly `plan: ExecutionPlanStub`.
    - Export `RunFailure` with readonly `ok: false`, readonly `error: LatticeRunError`, optional readonly `raw?: unknown`, optional readonly `partialOutputs?: Record<string, unknown>`, and readonly `plan: ExecutionPlanStub`.
    - Export `RunResult<TOutputs extends OutputContractMap> = RunSuccess<TOutputs> | RunFailure`.

    In `outputs/validate.ts`:
    - Import `type { StandardSchemaV1 }` from `@standard-schema/spec`.
    - Export `validateSchemaOutput<S extends StandardSchemaV1>(name: string, schema: S, value: unknown)` returning a promise of `{ ok: true; value: StandardSchemaV1.InferOutput<S> } | { ok: false; issue: { output: string; issues: readonly ValidationIssue[] } }`.
    - Call `schema["~standard"].validate(value)` and support both sync and Promise results.
    - Normalize issue paths by mapping Standard Schema path segments to `string | number | symbol` when possible.
    - Export `validateOutputMap<TOutputs extends OutputContractMap>(contracts: TOutputs, rawOutputs: Record<string, unknown>, plan: ExecutionPlanStub)` returning `Promise<RunResult<TOutputs>>`.
    - For `"text"`, require `typeof rawOutputs[name] === "string"`.
    - For `{ kind: "citations" }`, require `Array.isArray(rawOutputs[name])`.
    - For `{ kind: "artifacts" }`, require `Array.isArray(rawOutputs[name])`.
    - For Standard Schema, call `validateSchemaOutput`.
    - On first validation problem, return `ok: false`, `error.kind: "validation"`, `error.output`, `error.issues`, `raw: rawOutputs`, `partialOutputs` containing outputs validated before the failure, and the provided `plan`.
    - On success, return `ok: true`, typed `outputs`, `artifacts: []`, and the provided `plan`.
    - Do not call Zod `.parse`; use Standard Schema only.
